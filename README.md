## 🏗️ What I'm Building

<table>
<tr>
<td width="33%" valign="top">

### 🎯 TalentOS (ATS)
An AI-powered Applicant Tracking System built with a 3-service microservices architecture.
- Resume & CSV parsing with AI validation
- Admin-configurable pipeline stages (zero hardcoding)
- BullMQ + Redis job queues
- Team of 6 developers

</td>
<td width="33%" valign="top">

### 💬 Caspa Chat Platform
Production-grade real-time chat as a service.
- Socket.IO + Redis adapter for horizontal scaling
- WebRTC voice calling
- Multi-tenant widget via iframe + postMessage auth
- S3 file sharing, read receipts, typing indicators

</td>
<td width="33%" valign="top">

### 📚 LMS Platform
Scalable Learning Management System on GCP.
- 6+ microservices on Cloud Run
- Batch management, attendance, certificates (Puppeteer PDF)
- NM Analytics dashboard with cascading filters
- SSO external API for third-party enrollment
